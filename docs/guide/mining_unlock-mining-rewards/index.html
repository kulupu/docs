Unlocking Mining Rewards

The procedure for unlocking mining rewards is as follows:

  1. Visit polkadot.js.org/apps/#/explorer

note: make sure Kulupu is selected from the network dropdown (top-left corner of the page)

  1. In the main menu, click Developer -> Extrinsics
  2. At the top of Extrinsics is the "using the selected account" dropdown. Make sure it displays the account you're unlocking mining rewards for.
  3. Below is "submit the following extrinsic" dropdown. Default should be: system. Switch to rewards.
  4. On the same line, to the right is the "setLockParams" dropdown. Default value should be: setCode(code). Switch to unlock(target).
  5. A new menu appears below: "target: AccountID". By default, it uses the account from 3). Use it to switch to other accounts, if needed.
  6. Click "Submit Transaction", bottom-right corner of the page.
  7. Click "Sign and Submit".
  8. Enter account password to confirm.
